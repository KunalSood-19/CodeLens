# CodeLens AI 🔍

CodeLens AI is a smart code review and analysis tool that helps developers evaluate their code quality. It analyzes source code, calculates metrics like complexity, loops used, and lines of code, and provides suggestions for improvement.

## 🚀 Features
- Interactive code editor
- Automatic code analysis
- Code quality scoring
- Suggestions for improving code
- Code metrics visualization using charts
- Downloadable analysis reports
- Copy and download source code

## 🛠 Tech Stack
Frontend:
- HTML
- CSS
- JavaScript
- CodeMirror Editor
- Chart.js

Backend:
- Node.js
